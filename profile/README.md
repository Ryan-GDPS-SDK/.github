# Ryan GDPS
**Ryan GDPS** (Ryan GDPS SDK) is a powerful **framework** that you can create **mods** by installing the CLI.
## How to use it?
To create a mod, you need:
* **VS 2026** (or **2022** if you want)
* **CMake**

To use it, run this **command**:
```Bash
ryangdps new --type=cpp --name="MyMod" --category=none # You can select C++, Lua, JS, TS, Python and C# (CSharp) but i recommend C++ because is fast
```
It will say:
```text
Creating Project...
[##########] copying template... 100%
[##########] copying (182738/182738) items... 100%
Done! Project created!
```
it will create the entire **template**.

### What's next now?
Next: open **Visual Studio**, select **Open a Folder** (or **Open a Local Folder** or something else), select the **Folder** you created with the command and write your code in **main.cpp** or something else!
